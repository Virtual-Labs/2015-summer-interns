Requirements




For Ubuntu and Centos Systems


Follow the steps to install all the required plugins needed to run the experiment.
1) Click on Requirements to download the list of plugins needed.(Mandatory!)
2) Then click on Shell Script to download the script.(Depends on the above file!)
3) Press ctrl + alt + t(Terminal) and traverse to the Downloads folder.
4) Then with super user privileges type in sh shellscript.sh
5) The Packages will be downloaded and installed in your system.
6) Then go to the experiment site and allow if any permissions are asked.
7) If you don't have Java 3D Installed, download the following accordingly.
Java 3D 32 bit for 32 bit Systems.
Java 3D 64 bit for 64 bit Systems.
8) After downloading the zip files
9) You are ready to run the simulation!
All The Best and Enjoy!
