Basic Configuration




For Ubuntu and Centos Systems


Follow the steps to install all the required plugins needed to run the experiment.
1) Click on Requirements to download the list of plugins needed.(Mandatory!)
2) Then click on Shell Script to download the script.(Depends on the above file!)
3) Press ctrl + alt + t(Terminal) and traverse to the Downloads folder.
4) Then with super user privileges type in sh shellscript.sh
5) The Packages will be downloaded and installed in your system.
6) Then go to the experiment site and allow if any permissions are asked.
7) You are ready to run the simulation!
Additional Note: Few labs require Java 3D to be installed whose installation procedure is listed in Complete Configuration page.
All The Best and Enjoy!

For Windows Systems


Follow the steps to install all the required plugins needed to run the experiment.
1) You need Flash to run most of the labs. Click here to check whether flash is installed or not. If the version of the flash is displayed, you have the plugin installed. If not Click here to download the latest version of flash plugin.
2) You will also require Java to be installed for running many labs. For such labs, since Google Chrome doesn't support Java anymore, We recommend using Mozilla Firefox or MS-Internet Explorer.
3) If you have Java Installed, click the start icon and type "configure java". After getting the Java Control Panel, go to Security Tab and in Exception Site List add the link(for ex: http://cse11-iiith.vlabs.ac.in/ARM2/index.php?section=Experiment ) of the experiment to the list and click ok.
4) If you don't have Java Installed, download the following accordingly.
Java Runtime Environment 32 bit for 32 bit Systems.
Java Runtime Environment 64 bit for 64 bit Systems.
5) Next you have to set the path of java which will be like ("C:\Program Files\Java\jre6\bin") to the System Variables.Click here for further instructions to set the path.
6) You are ready to run the simulation. All the Best and enjoy!
Additional Note: Few labs require Java 3D to be installed whose installation procedure is listed in Complete Configuration page.
