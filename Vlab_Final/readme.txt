This File lists the steps to be done to install all the required plugins needed to run the experiment.

After Downloading the requirements.txt and shellscript.sh 

1) Press ctrl + alt + t(Terminal) and traverse to the Downloads folder.
2) Then with super user privileges type in sh shellscript.sh
3) The Packages will be downloaded and installed in your system.
4) Then go to the experiment site and allow if any permissions are asked.
5) You are ready to run the simulation!

All The Best and Enjoy!
